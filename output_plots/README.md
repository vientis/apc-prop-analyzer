# Output Plots Directory

This directory contains PNG plots generated by `char_plotter.py`.

## Contents
- Propeller characteristic plots showing Thrust, Torque, and Shaft Power vs RPM
- Files are named: `propeller_characteristics_<PROPELLER_NAME>_<SPEED>ms.png`

## Note
This directory is ignored by git (listed in `.gitignore`) to avoid committing generated plot files to version control.

All plots are automatically saved here by default when using the save option in `char_plotter.py`.
